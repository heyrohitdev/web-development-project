# 📝 To-Do List App

A simple To-Do List web application built using HTML, CSS, and JavaScript.

## 📌 About
This To-Do List helps users manage their daily tasks.
For example:
- What did you bring outside?
- Which subject did you study?
- Your daily activities.

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)
